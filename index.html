About

I'm a Pre-Sales Specialist who sits at the intersection of engineering and customer outcomes. My job is to take genuinely complex technical platforms — observability stacks, API security products, data pipelines — and make them click for the people evaluating them, buying them, and using them.

I've spent the last 3+ years at Rakuten SixthSense doing exactly that across three product lines: Full Stack Observability, Data Observability, and API Security. I've owned 20+ enterprise POCs, influenced $1M+ in pipeline, and at one point built a custom telemetry agent from scratch in under two weeks to save a deal that was about to fall apart.

I'm not a pure salesperson and I'm not a pure engineer. I'm the person who gets called when the customer is stuck, the demo isn't landing, the integration is broken, or nobody can agree on what the product actually does. I figure it out.

A GPT built from scratch — trained on Metal lyrics instead of Shakespeare. Inspired by Andrej Karpathy's "Let's Build GPT From Scratch" and implemented entirely in Python, adi-gpt is a character-level language model. It's the kind of project that starts as "I want to understand how this actually works" and ends with a model that generates surprisingly aggressive text.

The comparison demo shows the model at each stage of the tutorial — from basic bigram to full transformer blocks with attention, residual connections, and layer norm — so you can see exactly how each architectural addition improves the output quality.
